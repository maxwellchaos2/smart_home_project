# Голосовой Ассистент с Фильтрацией Шума 🎤

Умный голосовой ассистент с продвинутой системой фильтрации пустого звука и шума.

## ✨ Основные возможности

- 🎯 **Voice Activity Detection (VAD)** - Умная фильтрация тишины и шума
- 🔊 **RMS-анализ** - Вычисление среднеквадратического значения для точного определения речи
- ⚙️ **Настраиваемые пороги** - Адаптация под разные условия окружения
- 🌊 **Буферизация аудио** - Эффективная обработка потоковых данных
- 🔌 **WebSocket-интеграция** - Связь с сервером Vosk для распознавания речи
- 🐳 **Docker-контейнер** - Готовый к развертыванию сервер Vosk

## 🎛️ Настройка VAD

Система автоматически фильтрует:
- **Тишину** (RMS < 100) - Фоновый шум, отсутствие звука
- **Слабый шум** (100 < RMS < 800) - Шуршание, случайные звуки
- **Речь** (RMS > 800) - Человеческая речь, команды

### Настройка порога чувствительности

Отредактируйте значения в `lib/voice_assistant_service.dart`:

```dart
// Для тихой комнаты
const double voiceThreshold = 400.0;

// Для обычной комнаты (по умолчанию)
const double voiceThreshold = 800.0;

// Для шумной комнаты
const double voiceThreshold = 1200.0;
```

## 🛠️ Устранение неполадок

### Микрофон не работает на macOS
1. Откройте **Системные настройки** → **Конфиденциальность и безопасность**
2. Выберите **Микрофон**
3. Добавьте **Terminal** или **VS Code** в список разрешенных приложений

### Много ложных срабатываний
Увеличьте `voiceThreshold` до 1000-1200

### Пропускает тихую речь  
Уменьшите `voiceThreshold` до 400-600

## 🔬 Технические детали

### Алгоритм RMS (Root Mean Square)
```dart
double calculateRMS(Uint8List samples) {
  if (samples.isEmpty) return 0.0;
  
  double sum = 0.0;
  for (int sample in samples) {
    // Конвертация в знаковое значение
    final signedSample = sample > 127 ? sample - 256 : sample;
    sum += signedSample * signedSample;
  }
  
  return sqrt(sum / samples.length);
}
```

### Логика обнаружения голоса
- **Начало записи**: RMS превышает порог → включается запись
- **Продолжение**: Даже при тишине продолжаем запись (может быть пауза в речи)  
- **Остановка**: 15 чанков подряд ниже порога → останавливаем запись

### Тестовые значения RMS
- Тишина: ~70.7
- Шум: ~707.1  
- Речь: ~3535.5
